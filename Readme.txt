Description : 
-It is a classic Tetris game with multiple mods and features. JTetris2 is my second application made in java.

Features:
- Centralized high score, uploaded on a ftp server (done with ftp4j-1.7.2.jar 
  http://www.sauronsoftware.it/projects/ftp4j/ by Carlo Pelliccia).
- Customizable interface, input and sound.
- Multiple game modes (Campaign, Continuous, Challenge, Free Game).


JTetris2 Dependencies :
- In order to work the JTetris2.jar needs to be placed in the same folder as 
img, lib, Sound folders and config.dat, hof.dat files.
